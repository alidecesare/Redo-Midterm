FALL

p5.js/Java

I made this as an interactive assignment where you can click on the tree and the leaves will fall to the ground. I first had this project done in Processing through Java language, but adapted it to p5.js for simpler web availability.

MOON PHASES

p5.js

I created this for an assignment using buttons and sliders in p5.js. I enjoyed trying to figure out how to make it so the slider didn't go past certain distances, but also mapping that distance to the full length of the sketch to the moon phases could change. The way I made the moon phases occur with the slider is but attaching it to a circle that is the same color as the background so it looked like the moon image is being changed.

CATHARSIS

p5.js

I created this project as a way for students to destress. The little character on screen reacts to the microphone input, meaning when you scream at your computer, the character will grow and change colors, and when you stop screaming the character will scream back at you and shrink back to his original size.

Click on the image to be redirected to the sketch.
